# Smart-Irrigation-systems
An automated, real-time irrigation system developed on the STM32F407G Discovery (ARM Cortex-M4) board. The system optimizes water usage by monitoring soil conditions and automating actuator responses using a multi-tasking architecture.


Project Title & Tagline: Highlighting the STM32F407G and FreeRTOS integration.

Feature List: Covering ADC data acquisition, PWM actuation, and the RTOS scheduler.

System Architecture: A breakdown of the Sensor Task, Display Task, and Actuator Task to demonstrate your understanding of concurrency.

Hardware Pin Mapping: A clear table showing how you interfaced the moisture sensor and servo motor with the MCU pins.

Technical Stack: A quick-glance list of tools (C, HAL Drivers, STM32CubeIDE).
